# 企业简介与大连风景异形卡设计

**日期：** 2026-03-22

**目标：** 将首页当前 About 区块明确定位为“企业简介”，右侧采用企业文档中的正式简介内容并以目标站的稳重节奏呈现，左侧改为一张“大连风景”异形展示卡，整体建立更统一、更大气、更有企业站质感的版式关系。

---

## 设计背景

当前 About 模块虽然已经从重叠双卡调整为左右分栏，但内容组织仍然偏向“数据卡 + 功能点列表”，更像能力概览，而不是正式的企业简介。用户希望这一块回到目标站的阅读逻辑，即：

- 左侧承担视觉记忆点
- 右侧承担企业说明与信任表达
- 内容排布整齐，不拥挤，不做多余叠层
- 动效克制，整体稳重统一

同时，本次文案需要以 `C:\Users\Administrator\Desktop\大连博恒新技术有限公司20240419.doc` 中的公司简介为依据，而不是继续沿用概括式营销文案。

## 文档内容提炼

从已提取的文档正文中，企业简介的关键信息包括：

- 公司位于大连，具备明显的港口城市区位特征
- 专注钢铁行业连铸、连轧、冷轧等成套辊道设备及备品备件制造
- 拥有堆焊、喷焊、等离子堆焊等表面强化核心技术
- 长期与大连理工大学合作研发相关新技术应用
- 可按图制造，也可结合用户现场工况进行相应涂层设计
- 具备堆焊、喷焊、热处理、喷丸、装配等较完备制造能力
- 以质量、严谨、诚信为宗旨，强调高效率、低成本与高质量服务

这些信息适合整理成“1 段正式简介 + 3 条整齐能力条目”的结构，而不适合继续拆成零散的小卡片。

## 确认采用的方案

采用方案 A：

- 左侧为一张“大连海岸城市风景”异形卡片
- 右侧为“企业简介”正文区
- 整体版式参考目标站的左图右文节奏，不做交叠，不做复杂轮播
- 动效只保留轻量 reveal、分行淡入和克制的光感扫过

不采用的方向：

- 不做插画感过强的城市剪影方案，避免偏离真实企业气质
- 不做双层景深或轮播式信息块，避免抢掉简介阅读重心

## 模块结构

### 左侧视觉卡

左侧保留单一主视觉，但将现有普通图片卡调整为异形卡：

- 外轮廓语言参考首页异形按钮的斜切与圆角感
- 不是生硬矩形，也不做过度夸张的多边形
- 图片主体为“大连城市海岸 / 港湾 / 山海城市天际线”氛围
- 图片上只保留极少量信息点，例如城市名、小标签或轻量徽标
- 角标信息保留，但要弱化成辅助信息，不能压住图片本身

视觉目标是让左图成为“城市与企业气质”的承载面，而不是再堆数据。

### 右侧内容卡

右侧改为更接近目标站的正文式信息面板：

- 眉题使用“企业简介”
- 主标题强调“立足大连、服务钢铁装备与表面强化制造”
- 主体放置一段整理后的正式简介文本
- 正文下方增加 3 条整齐的企业能力短句，作为阅读收束

推荐的能力条目方向：

- 连铸、连轧、冷轧成套辊道设备及备件制造
- 表面强化核心技术与多工况材料应用经验
- 可按图制造，也可结合现场工况优化涂层方案

这 3 条应采用横向或纵向整齐的信息条，而不是重复现有 stats 小卡堆叠样式。

## 动效方向

动效参照目标站，保持克制：

- 模块进入视口时，左图和右文轻微上移显现
- 正文标题和段落分层淡入，不做夸张缩放
- 左图表面可增加一次低频、弱强度的光感扫过
- 异形卡片自身可有极轻微浮动，但不能像悬浮广告卡

重点是“高级感来自节奏和材质”，不是依赖明显运动。

## 内容与命名调整

- 本区块语义从“公司简介”明确收束为“企业简介”
- 原有以 stats 为主的表达弱化为辅助说明
- 右侧正文优先保留文档表达的专业性，不写成纯营销口号

## 风险与约束

- 不能破坏当前首页统一视觉体系
- 不能让异形卡片过于花哨，导致与首页 Hero 按钮语言不一致
- 不能让右侧正文变成大段挤压文字，需通过字重、行距、段前后距和信息条控制节奏
- 需要保留移动端自然回落能力，避免异形卡在小屏上失控

## 验收标准

- 首页 About 区块在视觉和语义上都能被识别为“企业简介”
- 左侧为单一异形大连风景卡，不再是普通矩形图卡
- 右侧出现正式企业简介段落，内容基于文档提炼
- 文案排版整齐、留白充分，不再出现“字堆在一起”的观感
- 动效保持稳重克制，接近目标站气质
- About 相关测试与首页构建验证通过
