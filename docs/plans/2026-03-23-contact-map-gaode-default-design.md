# Contact Map Gaode Default Design

**Goal**

将联系页面中的默认地图从通用嵌入切换为高德地图搜索页，并尽量把默认视野锁定在大连旅顺口区三涧堡街道附近，提升国内环境下的可用性与定位感。

## Root Cause

- 当前嵌入使用的是通用地图搜索 URL
- 在国内访问场景下稳定性和直观性不如高德
- 地址是村级地址，直接搜索容易不够精确，因此需要给地图一个固定范围辅助定位

## Chosen Fix

- 默认地图切换为 `www.amap.com/search`
- 使用：
  - `query` 传入完整地址
  - `city=210200` 限定在大连
  - `geoobj` 锁定三涧堡街道附近范围
  - `zoom` 提升默认聚焦程度
- 保留“在高德打开 / 在百度打开”外链作为兜底

## Evidence

- 公开可检索的 AMap 搜索链接格式显示其支持 `query`、`city`、`geoobj` 和 `zoom` 参数
- 通过地理查询获得三涧堡街道附近可用范围：
  - `121.2190244|38.9027021|121.2582175|38.9235801`
