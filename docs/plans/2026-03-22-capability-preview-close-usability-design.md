# Capability Preview Close Usability Design

**Goal**

修复首页能力图片预览层中“关闭按钮不好用”的问题。保留现有当前页预览机制不变，把关闭入口从单独悬浮小角标提升为更明确、更易点、更易理解的工具条关闭按钮。

## Root Cause

- 关闭逻辑本身已经存在，按钮和遮罩都绑定了关闭事件
- 本地页面 HTML 中关闭符号也能正常渲染
- 问题主要在可用性层面：
  - 按钮仅为右上角单个小图标
  - 视觉提示偏弱
  - 在大图预览场景下，点击目标不够明确

## Chosen Fix

- 将关闭按钮放入预览层顶部工具条
- 按钮增加可见文字“关闭”
- 保留 `×` 图标，但不再只靠图标表达
- 扩大点击热区，提升对比度与 hover/focus 反馈

## Scope

- 仅调整预览层顶部结构与关闭按钮样式
- 不修改打开/关闭逻辑主流程
- 不改变 hover 出图机制

## Testing

- 增加回归测试
- 验证组件包含工具条和关闭文字标签
- 验证样式包含工具条与关闭标签样式
