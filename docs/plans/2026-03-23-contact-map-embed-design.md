# Contact Map Embed Design

**Goal**

将联系页面中的地图占位块替换为真实地址地图，让用户在页面内直接看到公司地址所在位置，同时提供外部地图打开入口作为兜底。

## Chosen Direction

- 保留现有圆角地图容器
- 容器内部替换为真实地图 `iframe`
- 顶部增加简洁的信息条，显示“公司位置”和当前地址
- 右侧提供两个外部入口：
  - 在高德打开
  - 在百度打开

## Why This Direction

- 页面内直接看到地图，符合用户“直接把地址地图放上去”的要求
- 不依赖复杂 SDK，不引入额外 key
- 外部地图链接可作为加载不稳时的兜底方案
- 风格上仍能保持现在这套稳重、统一的蓝色样式

## Scope

- 仅调整 `ContactSection.astro` 的地图部分结构
- 补充对应的 `global.css` 样式
- 不改动联系表单与联系方式卡片

## Testing

- 新增回归测试
- 验证组件包含地图 `iframe`
- 验证组件包含高德和百度地图外链
- 验证样式包含地图工具条与 `iframe` 样式
