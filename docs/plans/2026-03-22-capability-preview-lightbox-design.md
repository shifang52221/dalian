# Capability Preview Lightbox Design

**Goal**

为首页“埋弧堆焊”卡片增加当前页大图预览层。用户鼠标悬停时先在卡片内部看到图片，点击卡片后在当前页弹出放大的图片预览，不跳转新页面，不破坏首页节奏。

## Chosen Direction

- 保留现有卡片内部 hover 出图
- 点击该卡片后打开当前页预览层
- 预览层采用整页遮罩 + 居中图片面板
- 继续使用整站蓝色体系，不做花哨弹跳动画
- 支持点击遮罩关闭、右上角关闭、`Esc` 关闭

## UX Notes

- 只给带图片能力的卡片加点击预览能力
- 卡片本体需要具备明确的可点击反馈
- 预览层出现时页面背景压暗并轻微模糊，突出图片主体
- 图片说明保持简洁，仅展示能力名称，不堆额外信息
- 打开时焦点切到关闭按钮，关闭后焦点回到触发卡片

## Scope

- 本轮仅处理“埋弧堆焊 / 埋弧肉盛”卡片
- 复用已有图片 `/images/capabilities/maihu.jpg`
- 不新增独立页面
- 不扩展到其他卡片

## Testing

- 新增交互结构回归测试
- 验证组件包含预览触发器和预览层标记
- 验证脚本包含打开/关闭与 `Escape` 处理
- 验证样式包含预览层开关态与 body 锁滚动规则
